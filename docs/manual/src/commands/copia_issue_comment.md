# copia issue comment

## copia issue comment

Add a comment to an issue

```
copia issue comment <number> [flags]
```

### Examples

```
  copia issue comment 12 --body "Investigating this now."
```

### Options

```
  -b, --body string   Comment body (required)
  -h, --help          help for comment
```
