## copia-cli repo clone

Clone a repository

```
copia-cli repo clone <owner/repo | URL> [<directory>] [flags]
```

### Examples

```
  copia repo clone my-org/my-repo
  copia repo clone https://app.copia.io/my-org/my-repo.git
```
