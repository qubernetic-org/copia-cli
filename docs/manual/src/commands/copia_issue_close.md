## copia issue close

Close an issue

```
copia issue close <number> [flags]
```

### Examples

```
  copia issue close 12
  copia issue close 12 --comment "Fixed in PR #7"
```

### Options

```
  -c, --comment string   Add a comment before closing
  -h, --help             help for close
```
