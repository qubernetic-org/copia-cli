# copia release upload

## copia release upload

Upload assets to a release

```
copia release upload <tag> <files>... [flags]
```

### Examples

```
  copia release upload v1.0.0 binary.tar.gz
  copia release upload v1.0.0 *.tar.gz *.zip
```

### Options

```

```
